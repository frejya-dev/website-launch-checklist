# Website Launch Checklist

A lightweight website QA tool built with HTML, CSS, and vanilla JavaScript.

The project is designed to help organize common tasks involved in launching a website, including content review, accessibility, SEO, analytics, performance, deployment, and post-launch verification.

## Features

- Website launch checklist organized by category
- Live progress tracking
- Responsive layout
- Semantic HTML
- Vanilla JavaScript (no frameworks)

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)

## Why I built this

Throughout my work as a web developer, I've found that successful website launches depend on having a consistent QA process. This project is an opportunity to practice modern JavaScript while building a tool inspired by real-world website launch workflows.

## Future Improvements
